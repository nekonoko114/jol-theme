# ACF（Advanced Custom Fields）設定ガイド - サブクエリ最適化版

ランキングページでライバー情報を表示するために、以下の ACF フィールドを設定してください。
**最新版では高速化のためサブクエリベースの実装を採用しています。**

## 必要なプラグイン

- Advanced Custom Fields (ACF) プラグイン

## 設定手順

### 1. フィールドグループの作成

1. WordPress 管理画面で「カスタムフィールド」→「フィールドグループ」をクリック
2. 「新規追加」をクリック
3. フィールドグループ名：「ライバーランキング情報」

### 2. 既存フィールド構成（設定済み）

以下のフィールドが既に設定されている前提で実装されています：

#### フィールド 1: クリエイター名

- **フィールドラベル**: クリエイター名
- **フィールド名**: creator_name
- **フィールドタイプ**: テキスト

#### フィールド 2: クリエイターアカウント

- **フィールドラベル**: クリエイターアカウント
- **フィールド名**: creator_account
- **フィールドタイプ**: テキスト

#### フィールド 3: 特徴

- **フィールドラベル**: 特徴
- **フィールド名**: feature
- **フィールドタイプ**: テキスト

#### フィールド 4: 配信開始時期

- **フィールドラベル**: 配信開始時期
- **フィールド名**: delivery
- **フィールドタイプ**: テキスト

#### フィールド 5: ギフト数（ランキング用）

- **フィールドラベル**: ギフト数
- **フィールド名**: gift
- **フィールドタイプ**: テキスト（数値として使用）

#### フィールド 6: 配信時間

- **フィールドラベル**: 配信時間
- **フィールド名**: delivery_time
- **フィールドタイプ**: テキスト

#### フィールド 7: アカウント URL

- **フィールドラベル**: アカウント URL
- **フィールド名**: account_url
- **フィールドタイプ**: テキスト
#### フィールド 8: Q&A（テキストエリア）

- **フィールドラベル**: Q&A
- **フィールド名**: liver_qa_text
- **フィールドタイプ**: テキストエリア
- **説明**:
  以下の形式でQ&Aを入力します。複数のQ&Aを設定する場合は、ブロックの間に `---` を挟んで区切ってください。

  **入力形式例:**
  ```text
  Q. 配信で一番大切にしていることは？
  A. 来てくれたリスナーさんが「今日も楽しかった！」と笑顔で寝れるような、最高の時間を共有することです。
  常に全力で楽しむことをモットーにしています！
  ---
  Q. 今後の目標を教えてください！
  A. もっと大きなステージに立って、みんなと一緒に新しい景色を見たいです！
  ```

### 3. 位置ルールの設定


「位置ルール」セクションで以下を設定：

- **投稿タイプ** が **ライバー** と等しい

### 4. 設定とスタイル

- **スタイル**: シームレス（推奨）
- **位置**: 通常（コンテンツエディターの後）

## データ入力方法

### ライバー投稿の作成

1. 「ライバー」→「新規追加」
2. ライバー名をタイトルに入力
3. アイキャッチ画像を設定（ライバーのプロフィール画像）
4. ACF フィールドにギフト数データを入力
5. 公開

### サンプルデータ例

```
投稿タイトル: サクラちゃんのライブ配信
クリエイター名: サクラちゃん
クリエイターアカウント: sakura_live
特徴: 歌とトークが得意
配信開始時期: 2023年1月
ギフト数: 1250
配信時間: 20:00-22:00
アカウントURL: https://example.com/sakura
```

## 確認方法

1. ライバーデータを 3 つ以上作成
2. ランキングページにアクセス
3. TOP3 とランキングリストに情報が表示されることを確認

## 最適化のポイント

### サブクエリベースの実装

- **高速化**: 直接 SQL クエリでランキング順序を取得
- **メモリ効率**: 必要な分だけデータを取得（LIMIT、OFFSET 対応）
- **キャッシュ機能**: 統計情報を 5 分間キャッシュしてパフォーマンス向上

### パフォーマンス向上

1. **TOP3 表示**: 最上位 3 件のみを取得
2. **一般ランキング**: 4 位以降を 7 件取得（3 件をスキップ）
3. **Ajax 読み込み**: ページネーションで追加データを動的読み込み

## 注意事項

- フィールド名（gift_count, monthly_gifts など）は正確に入力してください
- データが表示されない場合は、ライバー投稿が「公開」状態になっているか確認してください
- アイキャッチ画像が設定されていない場合は、デフォルト画像が表示されます
- 大量データでも高速表示が可能なサブクエリベース設計

## ランキング実績用のACF設定

ランキング実績（カスタム投稿タイプ：`ranking`）の各詳細ページで、3つのランキング（イベント、ダイヤモンド、配信時間）を表示するための関連ライバー設定を行います。

### 1. フィールドグループの作成・編集
1. WordPress管理画面で「ACF」（またはカスタムフィールド）→「フィールドグループ」を開きます。
2. 既にランキング実績用のフィールドグループがある場合はそれを選択し、無い場合は「新規追加」でフィールドグループ（例：`ランキング実績情報`）を作成します。
3. 位置ルールを以下のように設定します。
   * **投稿タイプ** が **ランキング実績** と等しい

### 2. フィールドの設定
以下の3つのフィールドを登録します。

#### フィールド 1: イベントランキング
- **フィールドラベル**: イベントランキングのライバー
- **フィールド名**: `event_ranking_livers`
- **フィールドタイプ**: **関連 (Relationship)** または **投稿オブジェクト**
- **投稿タイプでフィルター**: `ライバー`
- **返り値の形式**: 投稿オブジェクト または 投稿ID

#### フィールド 2: ダイヤモンドランキング
- **フィールドラベル**: ダイヤモンドランキングのライバー
- **フィールド名**: `diamond_ranking_livers`
- **フィールドタイプ**: **関連 (Relationship)** または **投稿オブジェクト**
- **投稿タイプでフィルター**: `ライバー`
- **返り値の形式**: 投稿オブジェクト または 投稿ID

#### フィールド 3: 配信時間ランキング
- **フィールドラベル**: 配信時間ランキングのライバー
- **フィールド名**: `delivery_time_ranking_liver`
- **フィールドタイプ**: **関連 (Rselationship)** または **投稿オブジェクト**
- **投稿タイプでフィルター**: `ライバー`
- **返り値の形式**: 投稿オブジェクト または 投稿ID
